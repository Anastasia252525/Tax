# Сервис автоматизации налоговых расчетов

Веб-сервис для управления налоговыми формами, сроками сдачи и уведомлениями.

## Функциональность

- Управление налоговыми формами (CRUD операции)
- Отслеживание сроков сдачи налоговых деклараций
- Система уведомлений о предстоящих сроках
- JWT аутентификация и авторизация
- Веб-интерфейс и REST API

## Технологический стек

- **Backend**: FastAPI, Python 3.11+
- **База данных**: SQLite (можно заменить на PostgreSQL)
- **Аутентификация**: JWT, bcrypt
- **Документация**: автоматическая OpenAPI (Swagger)
